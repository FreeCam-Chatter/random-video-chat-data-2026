# Random Video Chat Data 2026

Open dataset behind [The State of Random Video Chat 2026](https://freecamchatter.com/state-of-random-video-chat-2026/),
published by FreeCam Chatter.

Visit, engagement, geography, and referral-channel data for the 13 leading
random video chat platforms (OmeTV, Monkey, Flingster, CooMeet, Azar,
Emerald, Chatrandom, Bazoocam, Chatroulette, Chatspin, Camsurf, Shagle,
ChatHub), covering the February–April 2026 window.

## Files

| File | Contents |
|---|---|
| `data/platform-visits-feb-apr-2026.csv` | Visit volume and month-over-month change by platform |
| `data/engagement-metrics.csv` | Session duration, pages per visit, bounce rate, desktop share |
| `data/geography.csv` | Top traffic country and regional cohort by platform |
| `data/channel-mix.csv` | Organic search / direct / paid search traffic share |

## Source

Traffic and engagement figures are from Similarweb's Feb–Apr 2026 worldwide
panels. Search-volume trend figures cited in the full report are from Google
Trends. Full methodology: [`methodology.md`](./methodology.md).
